# jupyter-lab server

Jupyter Lab server running in docker.

## Supported Kernels

- Python
- Rust
- Go
- .NET
- 

## PREREQUISITES

- Docker

## BUILD

On MacOS or Linux

```bash
./scripts/fedora|ubuntu-build.sh
./scripts/fedora|ubuntu-run.sh
```

On Windows

```bash
./scripts/fedora|ubuntu-build.ps1
./scripts/fedora|ubuntu-run.ps1
```
